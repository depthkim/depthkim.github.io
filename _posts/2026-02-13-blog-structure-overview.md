이 글은 **depthkim.github.io** 저장소의 기본 구조를 빠르게 파악하기 위해 만든 예시 게시물입니다.

## 1) 현재 코드베이스 핵심 구조

- `_config.yml`: 사이트 전역 설정(제목, 설명, URL, 테마, 플러그인)
- `index.markdown`: 홈 화면 (`layout: home`)으로 최신 게시물을 목록 형태로 표시
- `about.markdown`: 정적 소개 페이지
- `_posts/`: 날짜 기반 기술 포스트 저장 디렉터리 (`YYYY-MM-DD-title.md`)

## 2) 화면에서 확인할 수 있는 구성

홈 화면에서는 아래를 확인할 수 있습니다.

1. 상단 헤더(사이트명, 내비게이션)
2. 최신 게시물 목록(제목, 날짜, 요약)
3. 푸터(테마 기본 정보)

게시물 상세 화면에서는 아래를 확인할 수 있습니다.

- 제목 / 날짜 / 카테고리·태그
- 본문 섹션 구조(H2, 목록, 코드 블록)

## 3) 코드 블록 렌더링 예시

```bash
bundle exec jekyll serve
```

```yaml
theme: minima
plugins:
  - jekyll-feed
```

## 4) 다음 확장 아이디어

- 카테고리/태그 아카이브 페이지 추가
- 검색 기능(클라이언트 사이드) 추가
- 댓글 시스템(Giscus 등) 연동
